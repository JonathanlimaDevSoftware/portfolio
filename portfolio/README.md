## 💻 Como Rodar o Projeto Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/JonathanlimaDevSoftware/my-portfolio.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd portfolio
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```
   ou
   ```bash
   pnpm install
   ```

4. Para rodar o projeto, execute:
   ```bash
   npm run dev
   ```
   ou
   ```bash
   pnpm dev
   ```

5. Abra seu navegador e acesse `http://localhost:3000` para visualizar o portfólio em funcionamento.
