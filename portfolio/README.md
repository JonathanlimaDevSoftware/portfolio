## Como Rodar o Projeto Localmente

Para rodar o portfólio localmente, siga as etapas abaixo:

1. Clone o repositório:

   ```bash
   git clone https://github.com/JonathanlimaDevSoftware/my-portfolio.git
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd my-portfolio
   ```

3. Instale as dependências:

   ```bash
   npm install
   ```

4. Execute o projeto:

   ```bash
   npm run dev
   ```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para visualizar o portfólio.
